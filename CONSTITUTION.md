# Lumicure Operating System Constitution

## Purpose

The Lumicure Operating System exists to make the company easier to understand, operate, scale, and improve.

## Principles

1. Keep the repository as the primary source of truth.
2. Record important decisions before they become invisible assumptions.
3. Prefer clear ownership, repeatable systems, and measurable outcomes.
4. Protect patient, customer, company, and partner confidentiality.
5. Improve the operating system continuously as Lumicure learns.

## Version

This constitution applies to Lumicure Operating System Version 1.0.
