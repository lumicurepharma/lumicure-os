## Purpose

This module defines the foundational principles, authority, governance, and boundaries of LOS.

## Principles

1. LOS is the source of truth for how Lumicure operates.
2. Strategy must be recorded before execution begins.
3. AI implements accepted decisions; it does not make strategy.
4. Institutional memory must survive team changes, tool changes, and time.
